**The Solution:**
I designed and implemented a production-ready integration using **Azure Web Apps** and **Python**.

*   **Key Features:** Automated daily scheduling, robust error handling, full audit logging to Azure Blob Storage, and secure secret management via **Azure Key Vault**.

**Outcome:**
*   Ensured 100% regulatory compliance.
*   Eliminated manual work and reduced human error.
*   Delivered a smooth production rollout with comprehensive architecture documentation.
*   **Tech Stack:** Azure Web Apps, Snowflake, Python (Flask), Docker, Terraform, GitHub Copilot.
